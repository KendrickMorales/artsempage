# artsemfab Website

Modern one-page portfolio site for a fabrication business focused on custom cage builds.

## Edit your content

- Update text in `index.html`
- Replace placeholder project visuals by swapping the `.project-media-*` blocks with real `<img>` tags
- Update contact links in `index.html`:
  - `mailto:you@example.com`
  - `https://instagram.com`

## Run locally

Open `index.html` directly in your browser, or run a static server if you prefer.

## Deploy to Cloudflare Pages (free)

1. Create a GitHub repo and push these files.
2. In Cloudflare dashboard, go to **Workers & Pages** -> **Create application** -> **Pages**.
3. Connect your GitHub repo.
4. Build settings:
   - Framework preset: `None`
   - Build command: (leave empty)
   - Build output directory: `/`
5. Deploy.

Your site will get a free `*.pages.dev` URL, and you can connect a custom domain later.
